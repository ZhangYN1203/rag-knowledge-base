RAG 智能知识库系统使用指南

欢迎使用 RAG（检索增强生成）智能知识库问答系统！本系统结合了向量检索技术和 AI 大语言模型，为您提供智能化的文档管理和问答体验。

一、系统功能概述

本系统提供以下核心功能模块：

1. 文档知识库管理：支持上传 TXT、PDF、Word（.docx）格式的文档，系统自动对文档进行文本提取、分段处理和向量化存储。上传的文档将作为 AI 问答的知识来源。

2. 智能问答：基于您上传的文档，系统使用 AI 大模型进行语义理解和回答。当您提问时，系统会先从知识库中检索与问题最相关的文档片段，然后结合这些上下文生成准确的回答。回答会以流式（SSE）方式逐字输出，并提供引用来源。

3. 对话管理：系统自动保存您的对话历史，您可以随时切换、查看或删除历史对话。

4. Prompt 模板管理：支持自定义 AI 提示词模板，使用 {{variable}} 语法定义变量，满足不同场景的问答需求。

5. AI 模型配置：支持切换 AI 提供商（Ollama 本地部署 或 OpenAI API），可调整对话模型、嵌入模型、API 地址、温度参数等。

二、快速上手

第一步：注册与登录

打开系统后，首先在登录页面点击"立即注册"，填写用户名、密码和邮箱完成注册。注册成功后会自动登录并进入聊天页面。

第二步：构建知识库

点击左侧导航栏的"知识库管理"，进入文档管理页面。点击上传区域选择文件，支持 TXT、PDF、DOCX 格式。上传后系统会自动处理文档并生成向量索引，处理完成后文档状态显示为"已处理"。

第三步：开始问答

回到聊天页面，在输入框中输入您的问题，按 Enter 键发送。系统会从知识库中检索相关文档，AI 会基于检索结果生成回答。回答以流式方式逐字显示，完成后会列出引用来源。

第四步：管理对话

左侧边栏显示所有历史对话记录，您可以点击切换对话，或点击"新对话"开始全新的问答。鼠标悬停在对话上可显示删除按钮。

三、高级功能

Prompt 模板管理：在 Prompt 模板页面，您可以创建和管理 AI 提示词模板。模板支持 {{variable}} 语法，系统内置了 RAG 问答的默认模板。

AI 模型配置：如果部署了其他 AI 模型或使用 OpenAI，可以在模型配置页面修改提供商、模型名称、API 密钥等参数。

四、技术架构

本系统采用前后端分离架构：
- 后端：Spring Boot 3.2.5 + Spring AI + Spring Security（JWT 认证）
- 前端：Vue 3 + TypeScript + Pinia + Element Plus
- 向量存储：使用 H2 数据库（开发环境）或 PostgreSQL + PGVector（生产环境）存储文档向量
- AI 模型：默认使用 Ollama 部署的 qwen2:0.5b（聊天模型）和 nomic-embed-text（嵌入模型）
- 文档解析：支持 PDF（Apache PDFBox）、Word（Apache POI）、TXT 格式

五、注意事项

1. 上传文档时请确保文件内容清晰可读，避免扫描件或图片格式的 PDF。
2. AI 回答的质量取决于文档内容的完整性和相关性，建议上传高质量的相关文档。
3. 如果使用 Ollama 本地部署，请确保已安装 Ollama 并已拉取所需的模型文件。
4. 系统样本文档仅供学习和测试使用，您可以根据需要删除或替换。
